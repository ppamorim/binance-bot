# binance-bot

Bot to automatically update your stop limit.

## Setup

 - On terminal run `touch env.toml` and paste the code below:

 ```toml
apiKey = "YOUR_API_KEY"
secretKey = "YOUR_SECRET_KEY"
 ```

 You can create these values on Binance dashboard.
